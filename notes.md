# Playwright Quick Commands

### 🚀 Running Tests
- `npx playwright test` : सर्व टेस्ट रन करण्यासाठी
- `npx playwright test <file-name>` : विशिष्ट फाईल रन करण्यासाठी
- `npx playwright test --project=chromium` : फक्त Chrome वर रन करण्यासाठी

### 🛠 Debugging & Tools
- `npx playwright test --ui` : इंटरअॅक्टिव्ह UI मोड उघडण्यासाठी (Best!)
- `npx playwright test --debug` : स्टेप-बाय-स्टेप डीबग करण्यासाठी
- `npx playwright codegen` : कोड रेकॉर्डर सुरू करण्यासाठी

### 📊 Reports
- `npx playwright show-report` : शेवटचा टेस्ट रिपोर्ट पाहण्यासाठी
